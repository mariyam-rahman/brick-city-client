
Brick City Lego Store
https://wondrous-sunburst-c479b5.netlify.app/


Brick City Lego Store is a simple full-stack Toy Marketplace website focused on Lego sets. It allows users to browse and purchase Lego sets, as well as add their own sets to sell. The website is designed to be visually appealing, with a unique and custom design.

Main Features
User Registration and Login: Users can create an account and log in using their email and password. The registration page includes fields for name, email, password, and photo URL.

Navbar and Footer: The navbar is present on all pages and includes the website logo, name, and links to Home, All Toys, My Toys, Add A Toy, Blogs, and User profile picture. The footer includes relevant information such as contact details, social media links, and address.

Home Page: The home page features a banner section with a slider or meaningful section, a gallery section showcasing relevant Lego set pictures, and a "Shop by Category" section. The "Shop by Category" section uses tabs to display sub-categories of Lego sets, such as Lego City, Lego Star Wars, Lego Architecture, and Lego Cars.

All Toys Page: The All Toys page displays a table or list of all Lego sets added by users. Each row includes information such as the seller (if available), toy name, sub-category, price, available quantity, and a "View Details" button. The page also includes a search system based on the toy name.

Single Toy Details: Clicking on the "View Details" button for a Lego set redirects the user to the Single Toy Details route, which provides detailed information about the set, including the picture, toy name, seller name, seller email, price, rating, available quantity, and a description. This route is private and requires user authentication.

Add A Toy Page: The Add A Toy page allows authenticated users to add their own Lego sets for sale. The form includes fields for the picture URL, name, seller name (if available), seller email, sub-category, price, rating, available quantity, and a detailed description. This route is private and requires user authentication.

My Toys Page: The My Toys page is accessible to authenticated users and displays a tabular form of the Lego sets they have added. Each row includes options to update or delete the toy. The update action allows users to modify the price, available quantity, and description of the toy. The delete action removes the toy from the list. Confirmation is required before deletion.

Blogs Page: The Blogs page includes informative blog posts where questions related to topics such as access tokens and refresh tokens, SQL and NoSQL databases, Express.js, Nest.js, and MongoDB aggregation are answered.

404 Page: The 404 page displays an interesting image or gif, along with a "Back to Home" button to redirect the user to the home page.
